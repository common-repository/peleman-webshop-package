A plugin for Peleman & partnered webshops, allowing for the sale of customizable products.

== Description ==
This plugin allows a WordPress/WooCommerce webshop to show and sell Peleman products. 
Some of our products require user content to either be uploaded and/or created in an online editor, which is made possible through this plugin.
